## Clone Blog 的原始碼

Step1. 自己到 Blog 的 Repository Clone 一份到本機端，它有提供滿多種方式，我就不特別講了。

Step2. 我自己是習慣用 VScode 來開發，當然如果你習慣用 Vim 或是其他編輯器也可以，接著在編輯器裡打開下載下來的原始碼。

Step3. 目錄結構如下，因為我也才剛開始接觸，對於裡面的目錄沒有那麼熟悉，等之後碰熟再回過頭補上目錄結構的說明好了。

![blog-categories](/static/images/blog-categories.png)

## 安裝相關套件

輸入`npm install` 來安裝相關的套件，安裝完成後，目錄結構裡便會出現 **node_modules** 這個資料夾。

## 部署至本機端

輸入 `npm start` 會開始部署至本機端，網址是 [http://localhost:3000](http://localhost:3000) ，如果想中斷部署的話，就按下 **control+C**。

![blog-deploy-localhost](/static/images/blog-deploy-localhost.png)

此時可以在瀏覽器的網頁上輸入 [http://localhost:3000](http://localhost:3000) ，就可以在本機端看見你的 Blog 了，之後有任何改動都會立即在本機端更新。
