# 為什麼要自己打造 Blog？

果然還是想自己從無到有建立自己的 blog，雖然上上個月才剛用 Wordpress 也架了部落格，用來放自己的 Podcast 筆記。但是我發現無法每天持續寫筆記，很容易就因為其他事情而怠惰，這樣一來我的 cloudways 的主機費用一樣在繳，顯得非常浪費。因此打算將 wordpress 上的文章搬到自己架的 blog 上，感覺講得好像超多文章，其實也才六篇文章 XDD

雖然說 blog 最重要的還是持續寫作，但身為一個前端工程師，還是想挑戰一下自己，看看能不能做出心中理想的 blog，同時也把打造的過程給記錄下來。

這次選擇使用這個 template：https://github.com/timlrx/tailwind-nextjs-starter-blog

原因是今年開始在學 React 所以決定選擇用 Next.js 來建立 blog。而 tailwind 則是我一直很想學，但公司負責的案子都還是使用 Bootstrap，平常也沒有在做 side project，所以就直接來實戰，邊做邊學。

# 對於這個部落格有什麼樣的需求？

### 整體風格

希望是簡單、乾淨的感覺。

### 頁面需求

主要會有四個項目，分別是首頁、文章列表、About 和文章頁面，稍微詳細一點的需求大概像這樣：

- **首頁**
  - about 簡短內容
  - 顯示文章列表（最新幾篇文章）
- 文章列表
  - 顯示文章列表
  - 有大標題、摘要、發文日期、閱讀時間、觀看次數、標籤、繼續閱讀。
  - 有頁碼

### 其他需求

- 有明暗模式可以切換
- 文章快速搜尋
- 頁面上方可以有進度條

以上是目前想到的需求
